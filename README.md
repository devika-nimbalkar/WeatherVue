#  WeatherVue- My Personal Weather Companion  

## 📌 Project Overview  
** WeatherVue** is a modern and responsive weather application that provides real-time weather updates for any location worldwide. The app fetches weather data using an API and displays key weather parameters like temperature, humidity, and weather conditions in an attractive UI.  

## 🚀 Features  
- 🔍 **Search for Any City** - Get weather details for any location.  
- 🌡 **Real-Time Temperature** - Displays the current temperature along with "feels like" data.  
- 💧 **Humidity & Conditions** - Shows weather descriptions like "Cloudy," "Sunny," or "Rainy."  
- 🎨 **Dynamic UI** - Background images change based on weather conditions (Hot, Cold, Rainy).  
- 📱 **Fully Responsive** - Works on all screen sizes.
